# ocr
## ocr with images preprocess
## 添加图片预处理的ocr简单实现
### python files contained in ocr dir
usage: 
      cmd: python ocrimg.py url language
      url:images url 
      language:optional(可选)
      such as：python ocrimg.py http://localhost:8080/test11.jpg

      cmd: python imgprocess.py img
      img：images location
